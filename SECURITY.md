# Security Policy

## Supported Versions
This repository is provided as a portfolio/showcase project.  
Security fixes are handled on a best-effort basis.

## Reporting a Vulnerability
If you find a potential security issue, please use GitHub private vulnerability reporting for this repository.

If private reporting is unavailable, open an issue without disclosing sensitive details and I will follow up privately.

## Scope
This project is for demonstration purposes and may not reflect production hardening standards.

## Response
I will review reports as soon as possible and, when applicable, patch or document mitigations.
