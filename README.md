# Fradel-Med (Showcase)

Tema WordPress customizado para a Fradel-Med, desenvolvido como projeto de **design + frontend + experiência de e-commerce** para exibição técnica no GitHub. Este repositório é de portfólio/demonstração.

---

## Visão geral

O projeto entrega um tema WordPress focado em:
- identidade visual da marca;
- experiência mobile/desktop;
- integração WooCommerce em nível de templates;
- ajustes finos de UI (header, busca, produto, carrinho e checkout).

---

## Principais pontos do tema
- Header responsivo com navegação, busca e ações rápidas.
- Home com hero visual e hierarquia clara de conteúdo.
- Página de produto com componentes personalizados e preço em destaque.
- Ajustes específicos para mobile (incluindo correções iOS).
- Estilo consistente para CTAs, formulários e navegação.
- Estrutura organizada para manutenção e evolução.

---

## Stack utilizada

- PHP

## Observações importantes
- Projeto com foco em apresentação técnica.
- Conteúdos podem estar simplificados para demonstração.
- Não há garantia de paridade com ambiente de produção.
- Não inclui credenciais, segredos ou dados sensíveis.

---

## Como visualizar
1. Suba um ambiente WordPress 6.x+ com WooCommerce.
2. Copie o tema para `wp-content/themes/`.
3. Ative pelo painel e navegue pela home/loja para ver os componentes.
